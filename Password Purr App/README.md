# Password Purr App

- Es una aplicación para dispositivos móviles hecha en MIT App Inventor que genera un sonido al presionar una imagen. Se accede mediante una contraseña.

Una vez colocada la contraseña correcta, la aplicación pasa a una segunda pantalla donde se escuchará un maullido al presionar la imagen de un gatito.
